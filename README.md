## Краткое содержание научной статьи

**Название:** An artificial intelligence model for heart disease detection using machine learning algorithms  
**Авторы:** Victor Chang et al. (2022)  
**DOI:** [https://doi.org/10.1016/j.health.2022.100016](https://doi.org/10.1016/j.health.2022.100016)

### Основная идея
Разработка системы предсказания сердечно-сосудистых заболеваний на основе алгоритмов машинного обучения с использованием языка Python. Авторы сравнивают несколько моделей и выбирают Random Forest как наиболее эффективный.

### Методология
- **Данные:** UCI Heart Disease Dataset (304 пациента, 14 признаков)
- **Предобработка:** Кодирование категориальных переменных, масштабирование числовых признаков
- **Модели:** Logistic Regression, KNN, SVM, Decision Tree, Random Forest
- **Оценка:** Accuracy, Confusion Matrix, Precision, Recall

### Результаты
- Лучшая модель: **Random Forest**
- Достигнутая точность: **~83%**
- Вывод: Ансамблевые методы показывают высокую стабильность и надежность в медицинской диагностике.

### Сравнение с данным проектом
В отличие от базового подхода в статье, в этом проекте использована **Нейронная сеть**, что позволило достичь:
- **Accuracy: 0.899**
- **ROC-AUC: 0.963**

Это демонстрирует преимущество глубокого обучения над классическими методами в данной задаче.
